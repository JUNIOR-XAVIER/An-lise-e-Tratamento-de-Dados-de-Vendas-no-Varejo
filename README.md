# Análise e Tratamento de Dados de Vendas no Varejo

Projeto de análise exploratória de dados da empresa **MaxLar Eletrodomésticos** utilizando Python e Pandas.

## Conteúdo
- `AnaliseLoja_Max.ipynb`: Notebook principal com a análise.
- `Vendas_SP.csv`: Dados de vendas em São Paulo.
- `gerentes_lojas.xlsx`: Informações dos gerentes.
- `vendas_tech.csv`: Dados de vendas de produtos tecnológicos.

## Tecnologias
- Python
- Pandas
- Jupyter Notebook

## Objetivo
Demonstrar técnicas de:
- Criação e manipulação de DataFrames
- Leitura e tratamento de arquivos Excel e CSV
- Filtros, seleções e transformações
- Agregações e análises com `groupby`
